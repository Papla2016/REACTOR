# PersonaVault (React demo)

Демо‑приложение с кабинетами гражданина, специалиста и администратора для управления персональными данными. Интерфейс использует React 18 и Tailwind CDN через Vite.

## Запуск на Windows
1. Установите [Node.js](https://nodejs.org/) версии 18 или новее. После установки убедитесь, что доступны команды `node` и `npm` в терминале PowerShell или CMD.
2. Склонируйте репозиторий и перейдите в каталог проекта:
   ```powershell
   git clone <URL-репозитория>
   cd REACTOR
   ```
3. Установите зависимости:
   ```powershell
   npm install
   ```
4. Запустите дев‑сервер:
   ```powershell
   npm run dev
   ```
   Vite выведет локальный адрес (например, `http://localhost:5173`). Откройте его в браузере.

### Сборка production
Для сборки оптимизированной версии выполните:
```powershell
npm run build
```
Готовые файлы появятся в папке `dist`. Проверить их можно командой:
```powershell
npm run preview
```
